# Angular Fast Editable Grid

High-performance editable grid in Angular 20.x  
✔ Always-editable cells  
✔ Virtual scroll (CDK)  
✔ Text / Textarea / Select / Date  
✔ Optimized for 500×25+  

## Features

- Virtualized rows for performance
- Always-editable inline cells
- No ngModel (faster change detection)
- OnPush strategy
- Schema-driven columns

## Run locally

```bash
npm install
ng serve
